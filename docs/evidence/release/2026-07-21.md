## Honest limitations

- This is a Python wheel, not a standalone Windows executable or installer.
- No code-signing certificate is configured and no publication or upload occurred.
- The validation used the same Windows 11 host as development, not a second clean machine.
- Live camera, calibration, emergency-hotkey, and real `SendInput` behavior were not activated by
