# Fresh source clone verification - 2026-07-21

## Scope

Verified a fresh Git clone and isolated dependency environment on the same Windows 11 host. This is
stronger than rerunning the workspace `.venv`, but it is not a second-machine or Windows 10 test and
does not include a webcam or real `SendInput` exercise.

## Revision and environment

- Git revision: `65b2e48a1c1826ff62ad97c3d7ed8690a08ebc89`
- Python selected by locked sync: CPython 3.11.15
- uv bootstrap: 0.11.30
- Install command: `uv sync --frozen --group dev`
- Installed from the committed `uv.lock`: 43 packages plus editable MEYES 0.1.0
- Clone location: a unique directory under the Windows user temporary directory

The host did not have a global `uv` launcher on `PATH`. To keep the clone isolated, a temporary
bootstrap virtual environment installed `uv>=0.8,<1`, then that executable performed the locked
project sync. This matches the documented prerequisite that judges install uv before setup.
